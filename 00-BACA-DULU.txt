⚡ QUICK START - PILIH SALAH SATU
═══════════════════════════════════════════════════════════════

OPTION 1: Linux/MacOS (Termudah)
────────────────────────────────
$ cd /workspaces/nest-lab-crud
$ bash start.sh


OPTION 2: Windows / Manual
────────────────────────────
$ cd /workspaces/nest-lab-crud/median
$ npm install
$ docker compose up -d
$ npx prisma migrate deploy
$ npm run prisma:seed
$ npm run start:dev


Lalu buka: http://localhost:3000/api

📞 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

❌ Port 3000 sudah dipakai?
   → Buka SETUP_GUIDE.md, cari "Port 3000 Already in Use"

❌ Database tidak connect?
   → Buka SETUP_GUIDE.md, cari "PostgreSQL Connection Failed"

❌ Migrations error?
   → Buka SETUP_GUIDE.md, cari "Migrations Failed"

❌ Module not found?
   → Buka SETUP_GUIDE.md, cari "Cannot find module"

Lihat SETUP_GUIDE.md untuk semua masalah & solusi!
